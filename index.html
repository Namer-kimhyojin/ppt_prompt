생성 프롬프트

                    
조합 및 최적화된 프롬프트가 실시간으로 노출됩니다.

                  

                  

                    
                    
                  

                

                
              


              

                
                
                
              

            

          


        

      


    

  



  

    

    

      

        

          
3D 샘플 미리보기

          
선택한 지도 3D 옵션의 예상 연출을 확인합니다.

        

        
      

      

        

          

          

            샘플
            

          

        

      

      

        
      

    

  


  

    

    

      

        

          
옵션 트리

          
상위 카테고리와 하위 옵션 그룹을 한 번에 보고 바로 이동합니다.

        

        
      

      

        

      

      

        
      

    

  


  

    

    

      

        

          
슬라이드 개별 프롬프트 설정

          
슬라이드별 선택 요소를 조정해 프롬프트를 다시 생성합니다.

        

        
      

      

        

          공통 색상 시스템과 높이 값 같은 공통 값은 현재 프롬프트 설계 탭의 공통 설정을 그대로 사용합니다.
        

        

        

      

      

        
        
        
      

    

  


  

    

    

      

        

          
슬라이드 구분 규칙 도우미

          
정규식을 직접 쓰지 않고도 자주 쓰는 구분 규칙을 만들어 입력창에 추가할 수 있습니다.

        

        
      

      

        

          `slide`는 본문 슬라이드 수에, `appendix`는 부록 수에 반영됩니다. 규칙은 반드시 `번호`와 `제목`을 잡는 2개의 캡처 그룹을 가져야 합니다.
        


        

          

            빠른 프리셋
            자주 쓰는 형식을 한 번에 추가합니다.
          

          

            
            
            
            
          

        


        

          

            규칙 만들기
            입력값을 고르면 규칙이 자동으로 만들어집니다.
          

          

            

              
              
본문 슬라이드인지, 부록인지 선택합니다.

              
            


            

              
              
MD에서 슬라이드 시작을 표시하는 `#` 개수입니다.

              
            


            

              
              
번호 앞에 오는 고정 단어를 입력합니다. 예: 슬라이드, 부록, STEP

              
            


            

              
              
01, 1, A1 같은 번호 형태를 고릅니다.

              
              

                
                
                
                
                
                
              

            


            

              
              
정규식 대신 실제 번호 예시를 넣어도 됩니다. 예: 01, A1, 1-1

              
            


            

              
              
번호 뒤에서 제목 앞까지 들어가는 구분자를 입력합니다. 예: `.`, `-`, `:`

              
            


            

              
              
아래 규칙이 실제 입력창에 들어갑니다.

              
            


            

              
              
현재 설정으로 인식될 예시입니다.
