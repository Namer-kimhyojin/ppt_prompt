진행 개요

페이지 유형과 전달 구조부터 선택하면 프롬프트 골격이 먼저 완성됩니다.

페이지 유형

표지, 목차, 본문, 맺음말처럼 슬라이드의 역할부터 먼저 정합니다.

필수

옵션 트리

상위 카테고리와 하위 옵션 그룹을 한 번에 보고 바로 이동할 수 있습니다.

입력

실제 본문 내용 입력

슬라이드 안에 실제로 반영할 제목, 수치, 구조, 키 메시지만 입력합니다.

표시 내용만 슬라이드 본문에 반영하고, 스크립트/맥락은 디자인 이해용으로만 전달합니다.

생성 프롬프트

선택한 항목에 따라 프롬프트가 실시간으로 조합됩니다.

입력

MD → Slide Prompts

모든 슬라이드에 공통으로 들어갈 스타일 가이드라인과 모델 지시문을 입력하세요.

또는 아래 입력창에 MD 내용을 그대로 붙여넣어도 됩니다.

한 줄에 하나씩 `type|regex` 형식으로 입력합니다. `type`은 `slide` 또는 `appendix`를 사용하고, 정규식은 `번호`와 `제목`을 캡처하는 2개 그룹을 포함해야 합니다.
