홍보용 이미지

작업 가이드
포스터, 카드뉴스, SNS 홍보이미지를 발표자료와 분리해 설계하는 전용 작업 탭입니다. 실제 화면 텍스트와 디자인 지시를 나눠서 프롬프트를 정리할 수 있습니다.

포스터, 카드뉴스, SNS 홍보이미지별 입력 항목과 프롬프트 미리보기, 설정 저장/불러오기를 제공합니다.

목적과 타깃을 먼저 정한 뒤 실제 노출 텍스트, 자산별 지시, 금지 요소 순서로 입력하면 정리가 쉽습니다.

필수 입력 누락과 비율/크기 오류를 바로 확인할 수 있고, 현재 설정을 JSON 파일로 저장하거나 다시 불러올 수 있습니다.
